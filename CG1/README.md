### Executando o Site

1. Baixe os arquivos
2. Navegue até o diretório `site` no arquivo `index.html` e usando o visual studio code use o comando: Alt + L Alt + O.
3. No site, vá até a barra rolante e clique no gif de cubo F giratório.
